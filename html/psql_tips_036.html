Psql Tip #036

Long options can take a value with or without the equal sign. 
laetitia:~$ psql --dbname laetitia
psql (14devel)
Type "help" for help.

laetitia=# \q
laetitia:~$ psql --dbname=laetitia
psql (14devel)
Type "help" for help.

laetitia=#
This feature is available since Postgres 9.1.
See Postgres documentation for more information.
